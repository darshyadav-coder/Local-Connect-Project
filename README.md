# Local Connect - Smart Services Booking System

A responsive service booking platform that connects users with trusted local professionals like plumbers, electricians, and cleaners. This project is a frontend-only web application that simulates real-world booking workflows using browser Local Storage.

---

## 🚀 Key Features

- Role-based authentication: user, provider, admin
- Signup with validation and duplicate email checks
- Login with demo accounts and registered users
- Dynamic service catalog from `js/data.js`
- Live search and categorized services
- Booking system with normal and emergency modes
- Emergency booking with extra charges and alerts
- Booking data stored in Local Storage (`allBookings`)
- User dashboard with booking status and feedback
- Provider dashboard for managing assigned bookings
- Admin dashboard with analytics and monitoring
- Contact/support form with message storage

---
